# BugTrackr

**BugTrackr** is a Java-based Command Line Interface (CLI) application for tracking software bugs. It allows users to report new bugs, update their status, and filter bugs by severity level.

## Features

- Report new bugs with a title and severity (`LOW`, `MEDIUM`, `HIGH`)
- Update bug status (`OPEN`, `IN_PROGRESS`, `RESOLVED`)
- List all reported bugs
- Filter bugs by severity
- Auto-incrementing unique Bug IDs

## Technologies Used

- Java
- OOP (Object-Oriented Programming)
- Java Collections (List, Enum)
- Scanner for user input
